## ⚡ ‎ Setupping

install dependencies :

```
  expo install
```

Change /src/service/api.ts :

```
const api = axios.create({
    baseURL: '' <= input your api
})
```

run project :

```
  expo start
```


## ❄️ ‎ Features
  - Login <br />
  - Create table (in draft) <br />
  - Create order in table ( with product amount )
  - Finish Order ( change draft to false )
